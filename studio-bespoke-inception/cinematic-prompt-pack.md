# Studio Bespoke Design - Cinematic Prompt Pack

**Status:** Pre-generation brief. No external generation, upload or credit spend is authorised by this document.  
**Selected territory:** The Ritual Plan  
**Sequence:** Mira - The decisive intervention

## Purpose

Create one short room film that helps a visitor understand the release created by opening the Mira kitchen, dining and living connection. It is not an invented before/after reconstruction and it is not a decorative hero background.

## Master specification

| Item | Requirement |
| --- | --- |
| Delivery role | Desktop scroll-scrubbed proof film; poster still on mobile and reduced motion. |
| Master | 16:9, 6-8 seconds, 24 fps preferred, no baked-in text or logos. |
| Camera | Slow, stable forward drift from a current Mira threshold/arch toward the joined kitchen-living space; level horizon, architectural-lens feeling, no handheld shake. |
| Beginning | A framed view that establishes threshold and controlled shadow; kitchen/living relation is partially withheld but truthful to the finished space. |
| End | The connected kitchen, dining and living space becomes legible in daylight; no impossible room geometry or new furnishings. |
| Light | Soft Dubai daylight; subtle natural exposure release only. No dramatic time-lapse, artificial lens flare, storm or night transformation. |
| Reverse safety | No people, boiling liquids, flames, moving screens, wind-blown curtains, flowing water or other time-directional events. |
| Text-safe zone | A quiet 25-30% edge on the source image, confirmed before generation; primary room evidence stays clear at centre. |
| Audio | None for the website interaction. An optional separate sound-design master may be planned later, but must not be needed for comprehension. |

## Required reference inputs

1. One rights-cleared high-resolution Mira image that shows the actual threshold and kitchen/living relationship.
2. One approved crop showing where copy can safely sit.
3. Written confirmation that the depicted materials, joinery, arch geometry and furniture are accurate to the completed Mira project.
4. If available, a second still that establishes the intended end framing. This improves continuity but is not mandatory.

## Source-image-to-video prompt

Use this only after the reference image is available and attached in the chosen generation tool.

```text
Use the supplied Mira interior photograph as the strict visual and architectural reference. Create a single continuous 7-second cinematic architectural shot inside this completed Dubai residence. Begin at the existing threshold/arched opening with a composed, slightly shaded view, then move the camera slowly and smoothly forward into the connected kitchen, dining and living space. The camera remains level and stable, with a measured dolly movement and restrained natural daylight opening across warm stone, plaster, timber, Calacatta surfaces and integrated joinery. Preserve the exact room geometry, materials, furniture placement, proportions and palette of the supplied photo. The centre of the frame must keep the kitchen-to-living connection clear; reserve a quiet text-safe edge in the composition. No people, no hands, no cooking, no steam, no moving screens, no candles, no flowing water, no wind, no dramatic sun rays, no lens flare, no cut, no morph, no invented demolition, no new objects, no stylised CGI. The shot must play convincingly in reverse for scroll scrubbing.
```

## Rejection criteria

Reject a generated clip if it changes room geometry, invents a renovation event, breaks material continuity, introduces time-directional motion, lacks a stable text-safe edge, or feels more like a luxury-real-estate reel than an explanation of a design decision.

## Website conversion contract

- Retain the original master and a poster frame.
- Extract a desktop WebP frame sequence only after FFmpeg/FFprobe analysis determines duration, FPS and a responsible frame count.
- Preload only the initial frame tier; lazy-load the rest near the Mira chapter.
- Use a static, caption-complete version for mobile, reduced motion, failed JavaScript and slow connections.
- Do not generate, upload or spend credits until the user approves the reference input and generation provider.
